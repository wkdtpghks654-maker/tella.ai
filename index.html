AI로 인해 상담 만족도가 떨어지는 경우는 없나요?

아닙니다. TELLA AI는 최신 LLM 모델을 적용해 자연스럽고 정확한 답변을 제공합니다. 실시간 모니터링과 지속 학습으로 직원보다 높은 응답 품질을 유지합니다.

추가 설치 비용이 있나요?

추가 설치 비용은 없습니다. 웹 기반 온보딩으로 바로 사용 가능하며, 필요한 연동은 전담 매니저가 무료로 지원합니다.

가격이 어떻게 되나요?

프리는 0원, 라이트는 월 29,900원, 베이직은 월 59,900원, 프로는 월 119,900원입니다. 각 요금제별 상세 기능은 위 요금표를 참고하세요.

기존 번호와 내부 시스템을 그대로 사용할 수 있나요?

네. 현재 사용 중인 대표 번호는 그대로 유지하며, 전화 라우팅과 내부 CRM/그룹웨어 연동도 지원합니다.

데이터 보안은 어떻게 관리되나요?

TELLA AI는 전 구간 암호화와 역할 기반 접근 제어를 적용하고, 주기적인 보안 점검을 통해 고객 데이터를 안전하게 보호합니다.
